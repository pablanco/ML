# MERCADO LIBRE TEST

Se implementa una app react y un middleware en node js de acuerdo a los requerimientos


## Para correr la aplicacion

1. Ejecutar `npm install` para instalar dependiencias.

2. Ejecutar `node server.js` para iniciar el servidor (queda escuchando en `http://localhost:3000/api/` ).

3. Ejecutar `npm run start` para iniciar la aplicacion frontend (navegar a `http://localhost:1234/`.).


## Posibles mejoras
